# BrPoliCorpus: Brazilian Political Corpus

by [Rodrigo Esteves de Lima-Lopes][def5] (Universidade Estadual de Campinas)

## Introduction
This file shows how to download the CSV files available for corpus compilation.

## How to download the files
Each of the links bellow will take you to a subcorpora spreadsheet with all the details regarding the nature and composition of the subcorpora. Just click in the download link that it might start automatically. 

1. [IndexProgrammes:][def]
  - Indexes all government proposals available.
2. [IndexInauguralSpeeches:][def2]
  - Indexes all Inaugural Speeches available.
  - Available in a single data frame (all the subcorpora)
3. [IndexFloorPlenaries:][def3]
  - Indexes all Floor Plenaries available.
4. [IndexCPI:][def4]
  - Indexes all Congress Committees of Inquiry available.
5. [IndexCommittees:][def6]
  - Indexes all Congress Committees (_not Inquiry_) available.
